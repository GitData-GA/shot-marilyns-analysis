GitHub Repository

All the resources are stored in the following GitHub repository branches:

  • main : This branch contains source code and data.

  • gh-pages : This branch contains files for the website of this project.

  • git-push-practice : This branch is to practice how to push files from Colab to GitHub directly.

  • paper-arXiv : This branch contains files for the paper in arXiv format. To update the paper, please follow the steps:

    1. Download the branch to edit the main.rmd file in RStudio:
      https://github.com/GitData-GA/shot-marilyns-analysis/archive/refs/heads/paper-arXiv.zip

    2. When updating, upload all files except the main-files folder to this branch.
